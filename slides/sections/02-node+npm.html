NodeJS and NPM

NodeJS is a JavaScript Server which runs on top of the V8 engine

npm is the built-in dependency management system for NodeJS

Installation of dependencies

The installation of npm modules is quite straigt-forward

Global dependencies:

            
npm install -g <package>

Local dependencies:

            
npm install <module>

Installation of persistent dependencies

If we want to have the dependencies installed on various systems (recommended) you have to save your dependencies in a package.json file.

        
npm init
        
    
        
npm install --save <module>
        
    
        
npm install --save-dev <module>
