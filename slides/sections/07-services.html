Services

Services are good for:

  • organize code
  • share code
  • share data


Angular services are:

  • Lazily instantiated
  • Singletons

Create a service

            
                angular.module('myModule', [])
                    .factory('myService', function(){
                        var service;
                        return service
                    });

Inject a service

            
                angular.module('myModule', [])
                    .controller('myCtrl', function($scope, myService){
                        $scope.status = myService.status();
                    });
