Resources

Definition

In common, a Web resource is anything that has an identity, typically the identity would be a Uniform Resource Identifier (URI).
http://yihongs-research.blogspot.de/2008/05/web-resource.html

Resources in Angular

  • Angular resources are built on top of the $http service
  • To use resources in Angular, we need to install the package angular-resource (retrievable via bower) and put the module ngResource into our module dependencies.
  • Angular's $resource service is a factory function for so-called resource objects.
  • A resource object is bound to a default URL and by default it gains access to the default HTTP methods GET, POST and DELETE.

Customizing the resource

Angular by default provides the following methods:

URL GET POST DELETE
/collection $query() $save()
/collection/id $get() $delete()

To get a more REST conform behavior of the resource objects, we can create our own service on top of $resource.
