Build-Management for static files with Grunt

http://gruntjs.com/

What is Grunt

Grunt is a "JavaScript Task Runner"

It is based on NodeJS

A lot of plugins are available in the npm repository:

  • processing of CoffeeScript files
  • processing of Less files
  • processing of Sass files
  • copying and concatenation of files
  • minification of JavaScript files
  • ...

Setup

        
npm install -g grunt-cli
        
    
        
npm init
        
    
        
npm install --save-dev grunt

Example: CoffeeScript compilation

        
npm install --save-dev grunt-contrib-coffee

Gruntfile:

            
module.exports = (grunt) ->

  grunt.initConfig
    coffee:
      dev:
        files:
          "dev/main.js": "src/**/*.coffee"

  grunt.loadNpmTasks "grunt-contrib-coffee"
            
        
        
grunt coffee
