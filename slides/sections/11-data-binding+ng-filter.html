Data Binding

Two-way data binding

example:

        
<body ng-app>

    <input type="text" ng-model="search"/>

    <h2 ng-bind="search"></h2>

    <ul>
        <li ng-repeat="item in items | filter: search">{{item.name}}</li>
    </ul>
    <button type="button" ng-click="myViewModel.heading = 'Demo'">Demo</button>

</body>
