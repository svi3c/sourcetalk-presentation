The test runner

In this workshop we use Karma as test runner.

Grunt has a plugin for running karma tests.

The task grunt dev is set-up to execute the tests everytime, a test or a source file changes.

The testing framework

The tests are written with a BDD-framework called Jasmine 2.0.

New tests for a section are prepared and skipped by default, so you can enable them to check whether your code is working as expected.

The tests are written in CoffeeScript.

  something = {}

  it "should behave a certain way", ->
