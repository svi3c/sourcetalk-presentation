AngularJS

https://angularjs.org

What is AngularJS?

It is an MVW framework for the browser

It provides us more power in the views by so-called directives

It is built on the two-way data binding concept with view models

It is very testable (if you use it as intended)

Installation

        
bower install --save angular

Two-way data binding

Two-way data binding

example:

        
<body ng-app>

    <h2 ng-bind="myViewModel.heading"></h2>

    <input type="text" ng-model="myViewModel.heading"/>

    <button type="button" ng-click="myViewModel.heading = 'Demo'">Demo</button>

</body>
