Filters

HTML:

            
                
    
                    
  • Selects a subset of items from array and returns it as a new array
    • 
                    
    
                    
  • {{ expression | filter : option}}
    • 
                

                

                    <input type="text" ng-model="myFilter"></input>
                    <div ng-repeat="item in items | filter:myFilter"><div>

Built-in Filters

            
                
    
                    
  • {{ date_expression | date : format}}
    • 
                    
    
                    
  • {{ number_expression | currency : symbol}}
    • 
                    
    
                    
  • {{ text_expression | uppercase | lowercase}}
